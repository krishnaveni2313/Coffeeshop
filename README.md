# Krishna Coffee Shop Landing Page

A simple, responsive landing page for **Krishna Coffee Shop**. The page is built with plain HTML, CSS, and JavaScript, so it can be opened directly in a browser without installing any extra tools.

## Features

- Warm coffee shop hero section
- Responsive navigation menu
- Coffee and snacks menu with prices
- Specials and opening hours section
- Contact details and contact form
- Mobile-friendly layout
- Lightweight static files

## Project Files

- `index.html` - Main page structure and content
- `styles.css` - Page design, colors, layout, and responsive styles
- `script.js` - Mobile menu and contact form interaction
- `README.md` - Project information
